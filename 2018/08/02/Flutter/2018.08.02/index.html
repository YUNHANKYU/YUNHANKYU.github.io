Flutter 공부 - Launcher Icon

2018-08-02

< Flutter 공부 > - Launcher Icon

1. pubspec.yaml 에서

dev_dependencies:
  flutter_test:
  sdk: flutter

//add flutter_launcher_icons package 
flutter_launcher_icons: "^0.5.2"

flutter_icons:
  image_path: "icon/icons.png"
  android: true
  ios: true

주석 아래 부분 입력 !

2. Terminal에 명령어 입력

Terminal 열기 > 프로젝트 폴더로 이동 > [ $ flutter pub get ] 입력 > [ $ flutter pub pub run flutter_launcher_icons:main ] 입력

3. Android 사진 확인

/프로젝트폴더/android/app/src/main/res/ 에서 사진 변경 여부 확인

4. Ios 사진 확인

/프로젝트폴더/ios/Runner/Assets.xcassets/AppIcon.appiconset 에서 확인

Good !!
